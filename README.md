# Grade Calculator in C

A simple command-line program written in C to calculate percentage and German grade.

## Features
- Input points and maximum points
- Calculates percentage
- Converts to German grading system
- Shows pass/fail status

## Example
Enter your points: 45  
Enter maximum points: 60  

Output:
Percentage: 75.00%  
German grade: 2.3  
Status: Passed

## How to run

gcc main.c -o grade
./grade
